# FibroDesign

A self-contained web app that generates fibromyalgia-related digital products — symptom trackers, self-care worksheets, quote cards, phone wallpapers, social media graphics, infographics, affirmation cards, and spoon theory explainers.

## Usage

Open `index.html` in any browser. No server required.

## Deploying to WordPress

Upload `index.html`, `css/`, and `js/bundle.js` to your hosting (e.g., via FTP to `public_html/tools/fibrodesign/`).

## What's Included

| Generator | Content |
|---|---|
| Symptom Trackers | Daily/weekly pain, fatigue, medication, sleep |
| Self-Care Worksheets | Triggers, energy conservation, flare-up prep |
| Quote Cards | 15 fibro-specific encouraging messages |
| Phone Wallpapers | 10 supportive reminder wallpapers |
| Social Media Graphics | 20 awareness posts across 6 content types |
| Infographics | Overview, symptoms, myths vs facts |
| Affirmation Cards | 30-card chronic illness set |
| Spoon Theory | Interactive explainer + static poster |

All designs can be downloaded as PNG or printed directly.
